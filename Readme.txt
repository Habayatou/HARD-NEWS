How to run the AfMAR Project using PHP and MySQL

1. Download the project zip file

2. Extract the file and copy AfMAR folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/Html)

4.Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with the name  afmar

6. Import afmardb.sql file(given inside the zip package in SQL file folder)

7. Run the script http://localhost/afmar
